Portfolio

I write small libraries. If a project implements something that may potentially be useful somewhere else, I'll factor it out.

My long term goal is to make Common Lisp a viable language for web development, and most of the code you'll see here reflects this. I've already done the ORM, a web framework, and a constellation of small libraries.

Thankfully, I'm not alone in this, and other people have done a lot of good work in this area: Eitaro Fukamachi, Nicholas Hafner, and Andrew Lyon. This portfolio wouldn't be what it is without them.
